Welcome to SystemHunt WebLabel release repository!

The app acts as an intermediary between your web application and the printers connected to the computer where the website is being accessed. It allows you to list available printers and send print commands directly to the printers.

WebLabel is a proprietary software. This repository offers a history of releases. For more information about how the software works and how to purchase a usage license, please visit [WebLabel website](https://systemhunt.com/weblabel).
